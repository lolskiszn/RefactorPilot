# RefactorPilot Pricing

## Proposed Model

### Open Source

- Free for public repositories
- Preview, reports, and read-only GitHub App checks

### Team

- Per private repository
- Includes safe apply, replay fixtures, and dashboard access

### Enterprise

- Annual contract
- Policy engine
- migration campaigns
- compliance reporting
- custom integrations

## Marketplace Split

- 70% author
- 30% platform

## Billing Notes

- usage metering should be metadata only
- no source code stored for billing
